# Sistema de Gestión de Inventario - Python

Este proyecto es un sistema de consola desarrollado en Python para la gestión de productos, aplicando estructuras de datos, modularidad y control de flujo.

## Estructuras de Datos Utilizadas
- **Diccionarios:** Para representar cada producto como una entidad con múltiples atributos (nombre, precio, stock).
- **Listas:** Para almacenar la colección completa de productos.

## Funcionalidades
- Registro de productos con validación de datos.
- Visualización de inventario en formato de tabla.
- Cálculo automático del valor total de activos.
